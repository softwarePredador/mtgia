## Mulligan & Strategy Notes

**Ideal hand:** 3-4 lands, 1 ramp spell, 1-2 cheap spells, curve under 4cmc
**Keep:** 3 lands + signet/top + interaction
**Keep:** 3 lands + Land Tax + any 2-drop
**Mulligan:** 0-1 lands, 6+ lands, all expensive spells, no red/white source

**Game plan:**
1. T1-3: Ramp, draw, set up topdeck control (Top, Scroll Rack)
2. T4-5: Cast Lorehold, protect with greaves/interaction
3. T6+: Chain cheap spell -> big spell to copy, generate value
4. Win via: copied big spells, Insurrection, Storm Herd, Approach, creature beatdown
