## Camada 2: Insights por Carta

### Pontos Fortes
- **Topdeck manipulation package:** Scroll Rack + Sensei's Top + Penance + Library of Leng = controle total do topo, sinergia direta com Lorehold
- **Ramp robusto:** 16 fontes de ramp, incluindo treasures (Smothering Tithe, Jeska's Will, Ancient Copper Dragon)
- **Board wipes assimétricos:** Obliterate e Jokulhaups resetam a mesa mas Lorehold permite recarregar do cemitério

### Pontos Fracos
- **Draw baixo:** Apenas 4 fontes de draw (Monument to Endurance, Furygale Flocking, Reforge the Soul, Season of the Bold). O deck depende de topdeck manipulation como pseudo-draw, mas em jogos longos vai ficar sem gas
- **Lands baixas:** 34 vs 36-38 recomendados. Com CMC médio de 3.68 e ramp abundante, o deck pode ter mãos iniciais inconsistentes
- **Sem recursão de cemitério:** 0 fontes de recursion vs 2-5 recomendados. Volcanic Vision é a única forma de reciclar spells do gy

## Camada 3: Mental Model do Deckbuilder

### Personalidade do deck
- **Estilo:** Criativo com foco em value engines
- **Tolerância a risco:** Média — inclui cards situacionais (Taunt from the Rampart, Penance) mas também staples (Sol Ring, Swords to Plowshares)
- **Foco principal:** Copy spells + treasures para value massivo

### O que este deck REVELA sobre como o jogador pensa:
O deckbuilder prioriza value engines (topdeck manipulation + copy spells) em vez de draw consistente. A estratégia depende de encontrar as peças certas via topdeck setup em vez de draw tradicional. É uma abordagem criativa mas arriscada — se o topdeck manipulation for removido, o deck perde o motor principal.
