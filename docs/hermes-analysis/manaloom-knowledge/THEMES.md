# Commander Themes — Referencia Completa de Deckbuilding por Tema

> Guia completo sobre como construir decks Commander por tema.
> O que define cada tema, quais cartas sao o "core package",
> como detectar o tema de um deck, e metricas ideais por tema.

## O que e um Tema de Deck?

Um tema e o **conceito central** que define como um deck tenta vencer.
Diferente de arquétipo (aggro/control/combo), o tema e o MECANISMO
especifico. Ex: "Tribal Elves" (tema) e um arquétipo aggro/combo.

Cada tema tem:
- **Enablers:** cartas que ativam/viabilizam o tema
- **Payoffs:** cartas que recompensam por seguir o tema
- **Core Package:** 8-15 cartas essenciais que definem o tema
- **Flex Slots:** cartas genericas que cabem em qualquer deck do tema

## Metricas Ideais por Tema

Cada tema tem METRICAS DIFERENTES. Um deck de Elves (20 ramp dorks)
precisa de menos ramp que um deck de Dragons (CMC medio alto).

### Ramp por Tema

| Tema | Ramp Ideal | Tipo de Ramp | Notas |
|:-----|:---------:|:------------|:------|
| Elfball | 20-30 | Dorks de mana | Cada elfo e ramp |
| Goblins | 12-18 | Dorks + Rocks | Goblin tribal tem ramp integrada |
| Dragons | 15-20 | Rocks + Ramp spells | CMC medio alto (4-5) |
| Spellslinger | 8-12 | Rocks + mana reducao | Paga pouco por spell |
| Voltron | 8-10 | Equipment cost reducers | Precisa de mana para equipar |
| Aristocrats | 8-12 | Rocks | Nao precisa de mana extrema |
| Enchantress | 8-10 | Mana dorks + enchant ramp | Encantamentos de ramp contam |
| Landfall | 15-20 | Land ramp spells | Mais terrenos = mais triggers |
| cEDH Combo | 20-30 | Fast mana + dorks | Precisa de mana TURNOS 1-3 |
| Mill | 10-12 | Rocks | Nao precisa de muita aceleracao |

## Como Detectar o Tema de um Deck

Dado um decklist, a deteccao do tema usa estas heuristicas:

### Passo 1: Contar tribos
- Se 20+ cartas compartilham um tipo de criatura -> TRIBAL
- Qual tipo tem mais cartas? Esse e o tema tribal

### Passo 2: Contar mecanicas
- 20+ instants/sorceries + 4+ spell payoffs -> SPELLSLINGER
- 15+ encantamentos + 4+ enchantress effects -> ENCHANTRESS
- 20+ artefatos + 4+ artifact payoffs -> ARTIFACTS
- 15+ token makers + 4+ token payoffs -> TOKENS
- 10+ +1/+1 counter sources -> COUNTERS
- 8+ sacrifice outlets + 4+ aristocrat payoffs -> ARISTOCRATS
- 10+ GY interactions -> GRAVEYARD
- 8+ reanimation spells + targets -> REANIMATOR
- 15+ landfall triggers + 10 land ramp -> LANDFALL

### Passo 3: Score de Confianca
- 3+ sinais fortes para o mesmo tema = CONFIRMADO
- 2 sinais fortes = PROVAVEL
- 1 sinal forte + 2 fracos = POSSIVEL
- Nenhum sinal = GENERICO (no theme, just good cards)

## Temas que Conflitam (Nao juntar)

- **Tribal + spellslinger:** quer criaturas, quer magicas. Salvo excessoes.
- **Voltron + tokens:** voltron quer 1 criatura forte, tokens quer muitas
- **Stax + reanimator:** stax para recursos, reanimator precisa de GY cheio
- **Mill + graveyard:** mill enche GY do oponente, graveyard enche o proprio
- **Aggro + group hug:** dar vantagem ao oponente enquanto tenta ser rapido

## Temas que Sinergizam (Juntar bem)

- **Tokens + aristocrats:** fichas para sacrificar -> drain
- **Spellslinger + spells matter:** obvio, mas funciona
- **Landfall + tokens:** Scute Swarm, Rampaging Baloths
- **Graveyard + reanimator:** obvio
- **+1/+1 counters + proliferate:** duplica counter gain
- **Artifacts + tokens:** treasure tokens contam como artefatos
- **Enchantress + pillow fort:** Ghostly Prison nao te atacam, voce encanta

## O ManaLoom e Temas

### O que ja funciona
- `detectedTheme` e `keepTheme` na otimizacao
- `coreCards` que nunca sao removidos
- Commander Reference Profiles com temas por comandante
- EDHREC themes via API
- Functional tags que identificam papeis

### O que precisa ser adicionado
- Deteccao automatica de tema a partir de decklist
- Metricas de qualidade por tema (nao genericas)
- Core packages documentados para cada tema
- Compatibilidade de temas durante otimizacao
- Sugestoes de swap que respeitem o tema dominante
- Score de "pureza tematica" (quantas cartas seguem o tema)
